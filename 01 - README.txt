Para rodar o sistema basta abrir o cmd ou terminal, redirecionar até a pasta do arquivo LucrosConnect\build\classes e colocar no terminal java Calculadora Server

Enquanto ele roda abra outro cmd ou terminal e redirecione igual ao passo acima. Feito isso, coloque java Calculadora Client
